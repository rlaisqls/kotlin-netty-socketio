# gribouille

[netty socketio](https://github.com/mrniko/netty-socketio)를 kotlin으로 재작성 및 개선해보는 프로젝트입니다.

## 목표

- netty 라이브러리를 직접 사용해봄으로써 낮은 추상화 수준의 자바 네트워킹 경험해보기
- java 코드를 kotlin으로 변환하며 언어의 특성 익히기
